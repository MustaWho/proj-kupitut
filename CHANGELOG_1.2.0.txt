Версия 1.2.0

Список изменений для комита:

1. Добавлен seed-скрипт app.seed_demo для создания тестовых данных.
2. Seed-скрипт создает три тестовых аккаунта:
   - admin / secret999
   - buyer / secret999
   - seller / secret999
3. Для тестовых пользователей добавлены валидные email:
   - admin@kupitut-shop.ru
   - buyer@kupitut-shop.ru
   - seller@kupitut-shop.ru
4. Исправлена ошибка ResponseValidationError из-за email с доменом .local.
5. Seed-скрипт создает 4 категории: Электроника, Дом, Книги, Одежда.
6. Seed-скрипт создает 6 тестовых товаров от имени продавца seller.
7. Seed-скрипт безопасен для повторного запуска: дубли пользователей, категорий и товаров не создаются.
8. При повторном запуске seed-скрипт обновляет данные тестовых пользователей, если они уже существуют.
9. В docker-compose.yml добавлен volume ./media:/app/media для сохранения загруженных изображений при работе через Docker.
10. Настроено сохранение пользовательских изображений вне контейнера, чтобы они не пропадали после пересборки.
11. Обновлена инструкция в README по запуску seed-скрипта через Docker.
12. Обновлена инструкция в README по тестовым логинам, email и паролю.
13. Обновлена PROJECT_DOCUMENTATION.txt с описанием тестовых данных и команды запуска seed-скрипта.
14. С главной страницы удалена подпись "Плитка до 4 строк" в блоке "Популярные товары".
15. Добавлены пояснения по проверке пользователей через /api/v1/users.
16. Добавлены пояснения по очистке Docker-базы и списка пользователей.

Команды для проверки:

Запуск проекта через Docker:

docker compose up --build

Создание тестовых данных:

docker compose exec api python -m app.seed_demo

Проверка списка пользователей:

http://localhost:8000/api/v1/users

Адрес сайта:

http://localhost:8080

Тестовые аккаунты:

- Администратор: admin / secret999
- Покупатель: buyer / secret999
- Продавец: seller / secret999

Проверки:

- Seed-скрипт проверен на тестовой SQLite-базе.
- Повторный запуск seed-скрипта не создает дубли.
- После двух запусков seed-скрипта остается 3 пользователя, 4 категории и 6 товаров.
