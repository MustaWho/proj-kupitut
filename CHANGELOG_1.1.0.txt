Версия 1.1.0

Список изменений для комита:

1. Добавлено отображение логина и ID пользователя рядом с аватаром в личном кабинете.
2. Добавлена загрузка аватара пользователя с сохранением файла по ID пользователя.
3. Добавлено отображение аватаров пользователей в отзывах.
4. Добавлена загрузка изображений товаров с сохранением файла по ID товара.
5. Добавлены папки media/avatars и media/products для пользовательских изображений.
6. При повторной загрузке аватара или изображения товара старый файл заменяется новым.
7. В форме регистрации и входа убраны заранее заполненные шаблонные данные.
8. В управлении товарами кнопки удаления и изменения получили текстовые подписи.
9. Продавцы и администраторы видят ID товаров.
10. Администратор может редактировать и удалять товар со страницы карточки товара.
11. В управлении добавлен поиск товара по ID для администратора.
12. Из управления добавлена возможность перейти к карточке товара.
13. В управлении добавлены вкладки администратора: товары, пользователи, продажи.
14. Добавлен поиск пользователей по ID, email и логину.
15. Администратор может открывать профили других пользователей.
16. Администратор видит ID, email, роль и статус пользователя.
17. Администратор может менять роль пользователя на покупателя или продавца.
18. Добавлена возможность блокировать и разблокировать пользователей.
19. Добавлен список заблокированных пользователей.
20. Добавлена история продаж: товар, покупатель, продавец, количество и сумма.
21. В форме добавления товара подписаны поля: название, категория, цена, скидка и описание.
22. Поле описания товара увеличено.
23. Описание товара в каталоге стало компактным и обрезается до двух строк.
24. Добавлена раздача пользовательских изображений через /media в FastAPI, Vite и Nginx.
25. Добавлена миграция для поля avatar_url у пользователей.
26. Добавлены типы Vite для корректной TypeScript-проверки.

Проверки:

- Бэкенд-тесты: 29 passed.
- TypeScript-проверка фронтенда: проходит.
- Endpoint /api/v1/sales проверен на пустой базе, ответ 200.
